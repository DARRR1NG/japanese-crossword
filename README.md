# Консольное приложение игры "Японский кроссворд"
2 курс, Программирование и основы алгоритмизации на С++

## :grey_question: Описание проекта
В рамках курсовой работы разработана программа на C++ для решения японских кроссвордов (нанограмм) с выбором уровня сложности. Японский кроссворд представляет собой логическую головоломку, где по числовым подсказкам, указанным слева от строк и сверху над столбцами, необходимо восстановить скрытое изображение, закрашивая соответствующие клетки. В отличие от классических кроссвордов, здесь требуется не знание слов, а дедуктивное мышление и внимание.

Программа построена с использованием объектно-ориентированного подхода. В ней реализованы два основных класса: 
Level, отвечающий за хранение данных об уровне сложности (размер поля, эталонное решение, подсказки для строк и столбцов), и JapaneseCrossword, который управляет игровым процессом, отрисовкой интерфейса, обработкой пользовательского ввода и проверкой победы.

Для визуализации используется консольный интерфейс с применением Windows API, что позволяет выводить цветные элементы: подсказки для столбцов отображаются красным цветом, для строк — синим, а само игровое поле имеет четкую структуру с границами и нумерацией. Справа от поля выводятся правила игры для удобства пользователя. Программа предлагает три уровня сложности с размерами сетки 5×5, 10×10 и 15×15, каждый из которых содержит заранее загруженное эталонное решение.

Особое внимание уделено обработке пользовательского ввода: реализована безопасная функция ввода целых чисел с проверкой диапазона, корректно обрабатываются специальные команды (выход по ESC, возврат в меню по вводу "0 0"). Каждый ход игрока проверяется на корректность, а после каждого действия программа автоматически сверяет текущее состояние поля с эталонным решением и оповещает о победе при полном совпадении.

В ходе разработки были применены современные подходы программирования: работа с динамическими структурами данных (векторы), форматированный вывод, обработка исключительных ситуаций и локализация интерфейса на русский язык. Программа прошла тестирование на всех уровнях сложности, демонстрирует стабильную работу и удобный пользовательский интерфейс.
